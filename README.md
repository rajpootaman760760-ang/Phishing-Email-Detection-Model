# 🎣 Phishing Email Detection Model

A cybersecurity project that detects whether an email is **Safe** or **Phishing** by analyzing suspicious patterns, keywords, links, and social engineering techniques.

This project demonstrates how phishing emails can be identified using machine learning concepts and text analysis.

## 🚀 Features

- Detect phishing emails
- Analyze suspicious keywords
- Detect suspicious URLs and domains
- Identify urgency-based phishing attempts
- Detect reward/money related scams
- Show confidence score
- Display suspicious signals
- Simple and interactive interface

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Machine Learning
- HTML
- CSS
- JavaScript

## 🧠 Machine Learning Models

The project compares:

- Naive Bayes
- Logistic Regression

The system classifies emails into:

✅ Safe Email  
🎣 Phishing Email

## ⚙️ How It Works

1. User enters an email message.
2. The system analyzes the email content.
3. It checks for phishing indicators:

- Suspicious keywords
- Fake links
- Suspicious domains
- Urgency language
- Money/reward related words

4. Based on detected patterns, it calculates phishing probability.
5. Final result is displayed as Safe or Phishing.

## 📌 Example

Input:

```
URGENT: Your account has been compromised!
Click here immediately to verify your identity
http://secure-login.xyz/account
```

Output:

```
🎣 PHISHING
```

Detected signals:

- Urgent language
- Suspicious URL
- Phishing keywords
